bool solve ( int **  ptr,
int  boardsize,
int  rpos,
int  cpos,
int  step 
)
Author
Riley Campbell
Description:
This function will solve any valid sized chessboard with any valid starting position using recursion. It will return false if the position it attempts to change to is not located on the board, or if its a position that has already been used and step back. If the position is valid, the array position will be set to the current step value. Next it will check to see if the the current step is equal to boardsize squared, and if so it will return true. If its neither true or false, it will resume the recursion process. If all the possible moves from the current position are invalid, it will set that position back to zero and back out till it can attempt a new route.
Parameters
[in,out]ptr- the 2d array that will hold the tour steps
[in]boardsize- the width and height of the chessboard
[in]rpos- the row position for the step
[in]cpos- the column position for the step
[in]step- the current number of moves the knight has made
Returns
true a solution was found
false a solution was not found
