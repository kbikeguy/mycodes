Program Information

This program will allow you to edit an SRT file, which contains the subtitles of a video file. The program will let you add frames, remove frames, update existing frames, print all frames, print specific frames, and save the result to a file of your choice.

The program will be run from a comand line. You will need to pass the file want to load as an argument. Once opened the program will allow you to edit the file in the manner stated above.

Once you are finished editing the file you can print the results to the screen to check your work. When you are happy with your results, you can save them to the original file, save them to a new file, or throw away all of your hard work.
